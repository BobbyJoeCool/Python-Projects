# Blackjack Version 1.0
Created by Robert Breutzmann

A simple blackjack game programmed in Python.  This program was created as a continuous learning for Python tecniques for intermediate coding and use of GitHub.

## Version 1.1 Update

- Added betting, player starts with a pool of 50 chips, minimum bet of 5 chips per hand.  If player wins, they double their bet.

## Version 1.0 notes

Basic text-line blackjack Game.

- Basic Dealer Logic
    - Stand on 17
    - Hit on Soft 17.
- Cards, Hands, Decks, Players are classes.
